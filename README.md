#                                     Sales Data Analysis Project

# Objective
The goal of this project is to analyze sales data to identify key business insights such as top-performing products, regional performance, and monthly 
trends.

# Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

# Key Analysis
- Revenue by product
- Sales by region
- Monthly sales trend

# Insights
- Identified highest revenue generating product
- Found best performing region
- Determined peak sales month
